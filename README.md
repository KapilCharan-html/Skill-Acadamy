# 🎓 SkillAcademy – Online Learning Platform

SkillAcademy is a **full-stack web application** designed to provide a structured and user-friendly online learning experience.  
It allows users to explore courses, enroll, and track progress, while administrators can manage course content efficiently.

---

## 🚀 Features

- 🔐 Secure User Authentication (Login/Register)
- 📚 Course Browsing with Categories & Filters
- 🎯 Course Enrollment System
- 📊 User Dashboard with Progress Tracking
- 🧑‍💼 Admin Panel for Course Management
- ⚡ REST API Integration (Frontend ↔ Backend ↔ Database)
- 📱 Responsive UI Design (Mobile + Desktop)

---

## 🏗️ System Architecture

User → Frontend → Backend (Java API) → MySQL Database → Response

---

## 🔄 Workflow

1. User accesses the platform
2. Registers or logs in
3. Frontend sends API request to backend
4. Backend validates user and interacts with database
5. User browses and filters courses
6. User enrolls in a course
7. Dashboard displays progress and enrolled courses
8. Admin manages courses and users
